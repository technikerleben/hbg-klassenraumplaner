# HBG Corporate Design Update

Diese Version der Klassenraumplaner-App wurde auf das offizielle HBG-Corporate-Design umgestellt.

## Eingesetzte Gestaltungsregeln

- Primärfarbe Rot: `#791D22`
- Sekundärfarbe Blau: `#245688`
- Tertiärfarbe Grün: `#3F6B4D`
- neutrale Hintergründe: `#F7F5F2` und Weiß
- semantische Farben: Warnung `#B8791E`, Fehler `#A62B2B`, Erfolg `#2E6E6E`
- Überschriften-Font-Stack: `Gagalin`, anschließend `Anton`, `Archivo Black`, `sans-serif`
- Fließtext-Font-Stack: `Glacial Indifference`, anschließend `Jost`, `Mulish`, `Helvetica Neue`, `Arial`, `sans-serif`

## Schriftdateien

Die Fontdateien werden nicht im Projekt gebündelt. Wenn Gagalin und Glacial Indifference auf dem jeweiligen Endgerät installiert sind, verwendet die App sie automatisch über `local(...)`. Andernfalls greifen die definierten Fallbacks.

## Logo

Da keine offizielle Logo-Bilddatei im Projekt liegt, verwendet die App einen klar markierten HBG-Logo-Platzhalter im CD-Rot. Für eine offizielle Veröffentlichung kann dort das echte Logo eingesetzt werden.
